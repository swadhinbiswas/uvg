Manage UVG's content-addressable store directly.

## Usage

```bash
uvg store {'<subcommand>'} [options]
```

## Subcommands

| Subcommand | Description |
|------------|-------------|
| `add` | Add a wheel to the store |
| `get` | Get a store object by hash |
| `remove` | Remove a store object |
| `list` | List store objects |
| `info` | Show store object details |
| `stats` | Show store statistics |

## Examples

### Add a Wheel

```bash
uvg store add path/to/package.whl
```

### List All Objects

```bash
uvg store list
```

### Filter by Name

```bash
uvg store list --name numpy
```

### Get Object Info

```bash
uvg store info abc123...
```


### Remove an Object

```bash
uvg store remove abc123...
```

**Note:** Objects referenced by active runtimes cannot be removed.

```bash
uvg store stats
```

## Why UVG?

UVG extends [UV](https://docs.astral.sh/uv/) to provide:

- **Content-addressable storage** — Packages stored once, shared across projects
- **Deterministic runtimes** — SHA-256 fingerprinting for instant reuse
- **Dependency intelligence** — AST import analysis, unused/missing detection
- **Workspace mode** — Monorepo discovery, shared dependency tracking
- **Security verification** — Hash verification, runtime integrity checks

## Quick Start

```bash
uvg init
uvg add requests numpy
uvg sync
uvg run python -c "import requests; print(requests.__version__)"
```

UVG can be configured via environment variables.

## UVG Variables

| Variable | Description | Default | Example |
|----------|-------------|---------|---------|
| `UVG_STORE` | Store location | `~/.uvg/store` | `/mnt/ssd/.uvg/store` |
| `UVG_PYTHON` | Python version | Current Python | `3.12` |
| `UVG_VERBOSE` | Verbose output | `false` | `true` |
| `UVG_QUIET` | Suppress output | `false` | `true` |

## UV Variables

UVG delegates to UV, so UV's environment variables also apply:

| Variable | Description | Default |
|----------|-------------|---------|
| `UV_INDEX_URL` | Package index URL | PyPI |
| `UV_EXTRA_INDEX_URLS` | Additional indexes | None |
| `UV_CACHE_DIR` | UV cache directory | `~/.cache/uv` |
| `UV_PYTHON` | Python executable | First in PATH |
| `UV_TLS_VERIFY` | TLS verification | `true` |
| `UV_NATIVE_TLS` | Native TLS | Platform default |
| `UV_LINK_MODE` | Symlink mode | `symlink` |

## Examples

### Custom Store Location

```bash
export UVG_STORE=/mnt/ssd/.uvg/store
uvg sync
```

### Verbose Output

```bash
export UVG_VERBOSE=true
uvg sync
```

### Private Registry

```bash
export UV_INDEX_URL=https://pypi.mycompany.com/simple
export UV_INDEX_USERNAME=user
export UV_INDEX_PASSWORD=pass
uvg add requests
```

### Offline Mode

```bash
export UV_NO_NETWORK=true
uvg sync
```
